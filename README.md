# Better_Student_Portal
做出學生真正需要的學生專區網頁

# 長榮高級中學學生專區
## 📌 專題簡介
這是我與小組成員在專題課程中完成的作品，目標是改善學校現有的學生專區網站。  
我們發現原本的網站資訊分散、介面不夠友善，於是決定重新設計一個更符合學生需求的版本。  
學校： 長榮高級中學  
製作時間：一學期(2025/9 ~ 2025/12)  
技術棧： HTML、CSS、JavaScript  


# 主要功能
## 多重倒數計時輪播
最重要的考試日期一目了然！

## 常用資訊快速連結
### 學習歷程
* 學生學習歷程平台
* 學習歷程檔案製作輔助平臺

### 我要學分
* 高中職線上系統
* 國中部線上系統
* 普通科畢業標準
* 線上選課==多元選修
* 線上選社
* 重補修

### 季節性資訊
* 運動週、運動會
* 校車資訊  
* 116班聯會

### 學好校規，保護自身權益
* 獎懲實施要點(校規)
* 國中部 獎懲實施要點(國中部校規)
* 性平


## 學長姐心得
透過彈出視窗（Modal）方式呈現學長姐的經驗分享
### 互動設計：
* 點擊卡片開啟文章
* 毛玻璃背景效果
* 滑順的淡入動畫
* 多種關閉方式（按鈕、點擊外部、ESC 鍵）
### 目前文章
* 職科證照檢定
* 毛毛的葉子
* 勸退指南

## 深色模式
* 一鍵切換淺色/深色主題
* 所有元件都支援深色模式
* 浮動按鈕，隨時可切換
